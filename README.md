
# Installation
```
$ git clone https://github.com/usmanalee/Home-Rent-Management.git
```
```
$ cd Home-Rent-Management
```
```
$ mv .env.example .env
```
```
$ composer install
```
```
$ php artisan migrate
```
```
$ php artisan db:seed
```
```
$ php artisan storage:link
```
```
$ php artisan serve
```
